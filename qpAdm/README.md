# qpAdm

```
grep 'summ' *.log
```

1. target: 被建模群体
2. rank: source 数
3. tail: p-value
4. dof: 可忽略
5. flag: 可忽略

**p-value 范围**
| p-value 范围 | 判读 | 含义 |
| :---: | :---: | :---: |
| p>=0.10 | 强通过 | 模型非常文件，right 怎么换都不容易倒 |
| 0.05<=p<0.10 | 可接受通过 | 常见于 ancient/right 稍多的情况 | 
| 0.01<=p<0.05 | 边缘通过/可疑 | 需要 right 敏感性分析 |
| 0.001<=p<0.01 | 明确拒绝 | 模型结构性不匹配 |
| p<0.001 | 强拒绝 | 不是统计问题，是模型错误 |
