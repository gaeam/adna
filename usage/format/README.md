# 格式说明

## EIGENSTRAT

### snp

> 标记 SNP 的信息（染色体、位置、变异等）

```
rs3094315 1 0.020130 752566 G A
rs12124819 1 0.020242 776546 A G
rs28765502 1 0.022137 832918 T C
1_842013 1 0.022518 842013 T G
1_846864 1 0.022720 846864 G C
```

**典型列**

1. SNP ID（位点名）

2. 染色体编号

3. 遗传距离：用重组率来衡量两个位点之间的远近关系，即两个遗传标记在减数分裂时通过重组被分开的概率，通常用厘摩（cM = centiMorgan）来表示，1 cM 近似于 1% 的几率在一次减数分裂中被分开，如果没有高精度的遗传图谱，可以直接填 0

4. 物理距离（碱基坐标）

5. 等位基因 1

6. 等位基因 2

### ind

> 个体的信息（样本 ID、群体、性别等）

```
Ind1 M Pop1
Ind2 F Pop1
Ind3 F pop2
Ind4 M Pop2
```

**典型列**

1. 个体名称

2. 性别

- M = male

- F = female

- U = unknown

3. 群体名称

### geno

> 基因型矩阵（样本在每个 SNP 上的基因型）

```
012000010010999099
000010000000999990
000110000000999099
000100000000999099
010000000100999990
201100000100999099
```

**储存方式**

- 每行对应一个 SNP，每列对应一个样本

**编码方式**

- 0 = 个体在该位点是参考等位基因纯合子

- 1 = 杂合子

- 2 = 另一个等位基因的纯合子

- 9 = 缺失数据

## PLINK

### bed

> 二进制文件，包含基因型信息，可以把 `fam` 中记录的个体和 `bim` 中记录的 SNP 位点联系起来

### bim

> 变异信息

```
1 rs3094315 0.020130 752566 G A
1 rs7419119 0.022518 842013 G T
1 rs13302957 0.024116 891021 G A
1 rs6696609 0.024457 903426 T C
1 rs8997 0.025727 949654 A G
1 rs9442372 0.026288 1018704 A G
```

**典型列**

1. 染色体编号

2. 变异标识符

3. 以摩尔根或者厘摩为单位的位置

4. 碱基坐标（从 1 开始计数）

5. 等位基因（通常是次要等位基因）

6. 等位基因（通常是主要等位基因）

### fam

> 存储样本的个体信息

```
Pop1 Ind1 0 0 1 -9
Pop1 Ind2 0 0 2 -9
Pop2 Ind3 0 0 2 -9
Pop2 Ind4 0 0 1 -9
```

**典型列**

1. 群体名称

2. 个体名称（不能为 0）

3. 父亲在数据集中的 ID（如果不存在于数据集中，写 0）

4. 母亲在数据集中的 ID（如果不存在于数据集中，写 0）

5. 性别

- 1 = male

- 2 = female

- 0 = unknown

6. 表型

- 1 = 对照组

- 2 = 实验组

- 0、-9 或非数字 = 缺失数据

## 索引

### fai

> 索引 `fasta` 

```bash
samtools faidx /Volumes/HardDrive/cattle_ARS-UCD2.0.fa
```

**典型列**

1. 序列名

2. 序列长度

3. 偏移量：序列在 `fasta` 中起始字节位置（会跳过描述行和换行符计算）

4. 每行碱基数

5. 每行字节数：包括换行符 `\n` 或者 `\t`

6. 最后一行碱基数
